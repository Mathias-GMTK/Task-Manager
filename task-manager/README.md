## Task Manager ##

# Aplicação simples de gerenciamento de tarefas (To-Do List) desenvolvida com React. Permite adicionar, concluir e remover tarefas de forma prática e intuitiva.

# Funcionalidades

- Adicionar tarefas
- Remover tarefas
- Marcar tarefas como concluídas
-

# Interface

- Design moderno com Tailwind CSS

src/
│── componentes/
│   └── MyButton.jsx
│── App.jsx
│── App.css


# Tecnologias Utilizadas
React (useState)
JavaScript (ES6+)
Tailwind CSS
